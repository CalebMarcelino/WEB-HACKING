WEB HACKING 1.0

Linguagens utilizadas: Bash e Python

Funcionalidades da ferramenta:

OSINT

Nesse processo a ferramenta faz todo processo de descoberta de informações sobre o nosso alvo, que provavelmente vai ser um web site, a ferramenta fara todo processo de reconhecimento... enumeração de informações como: números de telefone, emails, links relevantes, links de redes socias.

Descoberta de directorios

Nessa fase, a ferramenta procura descobrir outros directorios ou seja, possiveis caminhos padrões do site em que podera ser possivel descobrir informações sensiveis de sites e ate mesmo acesso a informações secretas do site.

Bruteforce attaque

Essa funcionalidade sera usada para atacar sites que não possuem proteção de ataques de força bruta, com esse ataque sera possivel hackear telas de logins de sites, telas de admin, usuarios e ate mesmo acesso api pagas.... caso as senhas sejam faceis de serem quebradas, a ferramenta fara diversos tipos de ataques de força bruta, ataque de força bruta por dicionário, ataque hibrido, randomico ou ate mesmo de senhas padrões.

hackear cameras de dispositivos

Essa funcionalidade permitira hackear cameras de dispositivos eletronicos como telemoveis, computadores por contra-inteligencia e links maliciosos, sera possivel capturar imagens em tempo real do alvo.



OBS: Ainda existem diversas outras funcionalidades que poderam ser implementadas na ferramenta WEB HACKING.
